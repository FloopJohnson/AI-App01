## üöÄ Installation Instructions

### Step 1: Download the Installer
1. You will receive a file named `AI-MaintenanceApp Setup 0.0.0.exe`
2. Save it to your Desktop or Downloads folder

### Step 2: Run the Installer
1. **Double-click** on `AI-MaintenanceApp Setup 0.0.0.exe`
2. If you see a Windows security warning, click **"More info"** then **"Run anyway"**
3. Click **"Next"** on the welcome screen
4. Choose installation location (default is recommended):
   - Default: `C:\Program Files\AI-MaintenanceApp`
   - Or click **"Browse"** to select a different location
5. Click **"Install"**
6. Wait for installation to complete (usually 1-2 minutes)
7. Click **"Finish"** to close the installer

### Step 3: Launch the Application
- The app should launch automatically after installation
- If not, look for **"AI-MaintenanceApp"** in your Start Menu or Desktop
- Double-click to launch

## üéØ First Time Setup

### Creating Your First Site
1. Click **"Add New Site"** in the left sidebar
2. Fill in the site information:
   - **Site Name**: e.g., "Main Distribution Center"
   - **Customer**: e.g., "Company Name"
   - **Location**: e.g., "Sydney, Australia"
   - **Contact Details**: Optional but recommended
3. Click **"Save Site"**

### Adding Equipment
1. Select your site from the main dashboard
2. Click **"Add Asset"** button
3. Fill in equipment details:
   - **Name**: Equipment description
   - **Code**: Equipment identifier
   - **Frequency**: Service interval in days
   - **Last Calibration**: Date of last service
4. Click **"Save Asset"**
